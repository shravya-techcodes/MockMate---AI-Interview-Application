# 🤖 MockMate AI – AI Interview Trainer

MockMate AI is a full-stack AI-powered interview preparation platform designed to help users practice technical and HR interviews through realistic mock interview sessions. The application will provide AI-generated questions, evaluate responses, and deliver personalized feedback to improve interview performance.

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite, HTML, CSS, JS
- **Backend:** Python, Flask
- **Database:** MySQL
- **API:** REST APIs
